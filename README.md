2d Platformer Game:
This is a 2D platformer game developed using Godot Engine. The game features a player character who must navigate through obstacles, avoid falling into holes, and reach the end point safely while collecting as many apples as possible.

- Smooth player movement
- Challenging platforms
- Collectible apples 
- Simple and clean 2D pixel design

  Controls:
  Move Left: Arrow Left
  Move Right: Arrow Right
  Jump: Space

  Built with:
  Godot Engine (2D Game Engine)

  Project Structure:
  player.tscn → Player character
  platform.tscn → Platforms
  apple.tscn → Collectible apples
  killzone.tscn → Dead zones (holes)
  game.tscn → Main game scene
